title: "FAST-CoExpress"
excerpt: "Fast, Adaptive, Scalable Tool for differential Co-Expression analyses"
collection: portfolio
---

The FAST-CoExpress framework models the joint distribution of $Y_1, Y_2$ conditional on $\mathbf{x}$ using a zero-inflated bivariate copula model where all parameters can be made dependent on subsets of covariates. Parameter estimation is achieved very quickly using frequentist framework.

[GitHub Repository](https://github.com/YenYiHo-Lab/FAST-CoExpress)
