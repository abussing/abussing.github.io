---
title: "FAST-CoExpress"
excerpt: "R package implementing a fast frequentist model for estimating differential co-expression"
collection: portfolio
---

The FAST-CoExpress framework models the joint distribution of two random variables using a zero-inflated bivariate copula model where all parameters can be made dependent on subsets of covariates. Parameter estimation is achieved very quickly using frequentist framework.

[GitHub Repository](https://github.com/YenYiHo-Lab/FAST-CoExpress)
